# CodeAlpha_LanguageTranslationTool
## Overview

This project is a GUI-based Language Translation Tool developed using Python. It allows users to enter text, select source and target languages, and translate text instantly using Google Translate services through the deep-translator library.

## Features

* User-friendly GUI using Tkinter
* Multiple language support
* Source and target language selection
* Real-time translation
* Copy translated text
* Text-to-Speech functionality
* Clear input and output fields

## Technologies Used

* Python
* Tkinter
* deep-translator
* pyttsx3

## Installation

Clone the repository:

git clone https://github.com/jayashreejbharathi-boop/CodeAlpha_LanguageTranslationTool.git

Navigate to the project directory:

cd CodeAlpha_LanguageTranslationTool

Install dependencies:

pip install -r requirements.txt

## Run the Project

python language_translator.py

## Project Workflow

1. Enter text in the input box.
2. Select source language.
3. Select target language.
4. Click Translate.
5. View translated output.
6. Optionally copy or listen to the translated text.

## Output

The application displays:

* Input text area
* Language selection dropdowns
* Translated text area
* Copy button
* Speak button
* Clear button
